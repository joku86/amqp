Archive-Connector
===================
Implementierung der persistirungsspezifischen Funktionalitäten für ein Archivierungsinstanz (HBase)

### Startargumente
Pflicht:  
-q "Queuname?Type,Quename2?Type"  
-r "Pfad zu der hbase-site.xml"  
-c "AMQP-Server details"  
-t "Tabelle"  
Optional:  
-p "Pfad für die Logs"
